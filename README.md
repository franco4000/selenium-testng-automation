# Selenium TestNG Automation Project

Proyecto de automatización QA desarrollado con Java utilizando Selenium WebDriver, TestNG y Maven.

## Tecnologías utilizadas

* Java 17
* Selenium WebDriver
* TestNG
* Maven
* ChromeDriver
* Git y GitHub

## Qué se automatizó

Se realizaron pruebas funcionales y end-to-end sobre la página:

[https://www.saucedemo.com](https://www.saucedemo.com)

## Escenarios probados

* Login exitoso
* Login inválido
* Compra completa E2E
* Eliminación de productos del carrito
* Logout
* Checkout vacío
* Usuario bloqueado
* Validación de total en checkout

## Funcionalidades utilizadas

* Assertions con TestNG
* WebDriverWait
* ExpectedConditions
* Validaciones positivas y negativas
* Automatización de flujos reales

## Ejecución

```bash
mvn test
```
